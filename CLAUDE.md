# sarza.ai

Portfolio and project management directory for web design and development work.

## Skills

These external skills enhance website design and testing work (installed at `~/.claude/skills`):

- `interview-first/` — interviews Jared with clarifying questions before building anything new, then confirms a written brief before any code is written
- `frontend-design.md` — Anthropic's official frontend design skill for creating high-quality UI components and layouts
- `web-design-guidelines.md` — Vercel's web design guidelines for modern, performant web experiences
- `webapp-testing.md` — Playwright-based webapp testing for automated quality assurance

Not currently installed (candidates if we want broader style/palette coverage later):

- `design-loop` — Multi-page website builder with baton pattern for design consistency across pages ([jezweb/claude-skills](https://github.com/jezweb/claude-skills/blob/main/plugins/frontend/skills/design-loop/SKILL.md))
